## 사용법

1. Nano Banana에 공통 프리픽스 + 개별 프롬프트를 합쳐서 입력
2. 생성된 이미지를 `/public/images/portraits/{generalId}.png`로 저장
3. `GeneralPortrait` 컴포넌트가 자동으로 로드 (파일명 = generalId)
4. 특수 매핑이 필요한 경우 `GeneralPortrait.tsx`의 `PORTRAIT_IMAGE_MAP`에 추가
